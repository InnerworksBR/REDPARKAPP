# 🚗 RED PARK

Sistema mobile first para gerenciamento de mensalistas de estacionamento.

---

## 📌 Objetivo

O RED PARK foi desenvolvido para facilitar:

- Cadastro de mensalistas
- Cadastro de veículos
- Gestão de mensalidades
- Controle de acesso/check-in
- Dashboard administrativo
- Integração com pagamentos online

---

## 🛠️ Stack Tecnológica

- Next.Js — Frontend / Backend
- Supabase — Banco de Dados, Auth e Storage
-  Mercado Pago — Sistema de Pagamentos
-  Vercel— Hospedagem 

---

## ⚙️ Funcionalidades

### 👤 Mensalista
- Cadastro
- Login
- Gestão da mensalidade
- Pagamento online
- Check-in digital

### 🛠️ Admin
- Dashboard financeiro
- Gestão de mensalistas
- Controle de inadimplência
- Baixa manual

---

## 📱 Mobile First

Sistema desenvolvido com foco total em dispositivos móveis:
- Interface moderna
- Navegação simples
- Layout responsivo
